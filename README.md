# CareerIntel Pro

A professional AI-powered job discovery and CV analysis platform.

## Features

- **Global Job Search**: Scrape jobs from multiple sources worldwide
- **CV Analysis**: AI-powered CV quality assessment and improvement suggestions
- **Smart Matching**: Match jobs to your skills and experience
- **7 Professional Themes**: Customizable UI with multiple themes
- **Real-time Insights**: Market trends and skill demand analysis
